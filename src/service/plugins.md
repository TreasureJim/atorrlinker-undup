# Location 
The plugins will be held in `ATORRLINKER_PLUGS` environment variable. Each plugin will have its own folder. 
Eg. For plugin named "elephant" it will be placed in `ATORRLINKER_PLUGS/elephant/main.py`.

# Requirements
If a python requires any extra packages it needs to be written in its `requirements.txt` file in the same folder as the plugins main python script.
